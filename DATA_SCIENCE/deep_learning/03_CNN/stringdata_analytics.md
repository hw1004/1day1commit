# 문자열 데이터 수치화
## 데이터 전처리
1. 정규화로 한국어만 남기기
   - `import re`: 정규식 함수
   - `re.sub('[^가-힣ㄱ-ㅎㅏ-ㅣ\\s]','',정규화할 문장)`: 한글과 공백제외 모두 제거한다.
2. 형태소 분석기로 어간 추출
   - `from konlpy.tag import Okt`
   - `okt = Okt()`
   - `okt.morphs(형태소 단어 단우이로 나눌 문장, stem = True)`
     - 텍스트를 형태소 단위로 나눈다.
     - `stem`: 각 단어에서 어간 추출
3. 불용어 제거
4. 문장 인덱스 벡터로 전환
5. 패딩 처리
